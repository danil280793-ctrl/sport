# 🌟 Ежедневный Трекер Привычек для Ребенка

## ✨ Основные возможности

### 🏋️ Физическая активность
- Комплекс упражнений, разбитый по дням недели.
- Адаптированная нагрузка для детей (приседания, отжимания от стены, планка и т.д.).
- Анимированные карточки с описанием техники выполнения.

### 🧠 Нейрогимнастика
- Упражнения для мозга на каждый день недели.
- Тренировка памяти, внимания, координации и мелкой моторики.
- Встроенный таймер для каждого упражнения.

### 🎮 Геймификация и Мотивация
- **Счетчик серий (Streak):** Отслеживает, сколько дней подряд ребенок занимается.
- **Система достижений:** Бейджи за первые шаги, недельную серию и общее количество выполненных заданий.
- **Конфетти-эффект:** Красивый салют при выполнении дневной нормы.
- **Общая статистика:** Главный экран показывает сумму всех достижений.

### 📱 Технические фишки
- **Работает без интернета:** Прогресс сохраняется в памяти браузера (`LocalStorage`).
- **Адаптивный дизайн:** Отлично выглядит на телефоне, планшете и компьютере.
- **Темная тема:** Автоматически подстраивается под настройки системы.
- **PWA Ready:** Приложение можно добавить на главный экран смартфона.
